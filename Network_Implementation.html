Hospital Network Implementation

implementing the secure and scalable Network for the hospital

Project Overview

This project focused on designing and implementing a secure, scalable, and efficient network infrastructure for a modern hospital spanning three floors. The design supports fast communication, secure data access, and seamless operation across medical, administrative, and guest areas.

Key Implementation Areas

  • Network Design: Built in PNETLab using Layer 3 switches, VLANs, and HSRP for redundancy.
  • Server Configuration: Implemented MCSA with Active Directory, DHCP, Group Policies, and domain services.
  • Security: Deployed FortiGate firewall with FSSO for internal users and Captive Portal for guests.

Tools Used

  • PNETLab for network simulation and topology testing.
  • VMware for virtualized Windows Server setup.

Core Network Topology

  • 2 Core Switches for inter-VLAN routing and high availability.
  • 6 Access Switches distributed across departments and floors.
  • VLAN segmentation for Management, IT, Doctors, Nursing, Finance, Guests, CCTV, and VoIP.

Infrastructure Highlights

  • Over 400 data points and structured cabling for devices like PCs, IP phones, and printers.
  • Wi-Fi Access Points deployed based on user density and hospital layout.
  • Domain structure with OUs for role-based access and GPO enforcement.
  • Roaming profiles and home folders for staff mobility.

Security Features

  • FortiGate firewall with LDAP-integrated FSSO for internal users.
  • Captive Portal for guest VLAN to ensure safe internet access.
  • Application control, antivirus, and web filtering profiles for network protection.
