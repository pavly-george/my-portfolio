Smart Bank Network Implementation

This project involved designing and implementing a secure network for a bank across four floors. The network's architecture focused on providing efficient, scalable connectivity with features such as VLANs, DHCP, OSPF routing, and switchport security. The project ensured compliance with industry-standard security protocols, providing a highly reliable and secure banking network infrastructure.

Key Features:

  • Structured cabling across four floors
  • VLAN configuration for network segmentation
  • DHCP for dynamic IP addressing
  • OSPF routing for inter-floor communication
  • Switchport security for endpoint protection
