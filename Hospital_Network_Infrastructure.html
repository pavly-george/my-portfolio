Hospital Network Infrastructure Design

Designing a Future-Ready Healthcare Environment

Project Overview

This project focused on designing a secure, scalable, and efficient network infrastructure for a modern hospital spanning three floors. The design ensures fast and reliable communication between departments, secure patient data handling, and seamless operations across all medical and administrative units.

Key Features

  • AutoCAD used for detailed infrastructure planning.
  • Location planning for data outlets and telecom rooms (MDF, IDF).
  • Structured cabling system using Ethernet.
  • Network segmentation and high availability following HIPAA standards.
  • Wireless Access Points placed strategically for full coverage.
  • CCTV Cameras with NVR for Security of the hosital.

Floor Plans

  • First Floor: Reception, management offices, main server room.
  • Second Floor: Operating rooms, ICUs, scan rooms, and IT room.
  • Third Floor: Patient rooms, recovery units, maternity areas.

Network Highlights

  • Over 300 data points (faceplates) for VoIP, printers, and medical devices.
  • Ceiling Access Points for staff, patients and Guests connectivity.
  • Redundant network closets (MDF & IDF) for each floor.
