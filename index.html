My Learning Journey

Networking Fundamentals

I started my journey by learning the basics of networking, including understanding network topologies, IP addressing, and subnetting. These fundamentals laid the foundation for my career in network design and security.

Introduction to Cisco and CCNA

As part of my learning, I delved into Cisco technologies and began preparing for the Cisco Certified Network Associate (CCNA) certification. I learned about VLANs, routing protocols like OSPF, and switchport security.

Microsoft Certified Solutions Associate (MCSA)

In pursuit of the MCSA certification, I studied Windows Server and the various components of Microsoft technologies. This training has equipped me with essential skills in server management, Active Directory, and system administration.

Fortinet Technologies

I gained knowledge in Fortinet security solutions, specifically FortiGate and FortiManager. This training has helped me understand network security best practices, including firewall management and centralized policy management.

Cloud Technologies

I have explored various cloud platforms, with a particular focus on Microsoft Azure. My learning encompassed Azure deployment models, services like Azure Virtual Machines and Azure App Services, and management tools that enable efficient cloud-based solutions and infrastructure management.

AutoCAD and Revit for Network Design

Learning tools like AutoCAD and Revit allowed me to design network layouts with precision. This skill set helped me work on projects like the Smart School and Smart City Infrastructure designs.

Learning GNS3 and PNetLab

As a hands-on learner, I’ve been using emulator programs like GNS3 and PNetLab to simulate network environments, allowing me to experiment with real-world scenarios.

German Language Course (KOMET Institut)

I am currently enrolled in a German language course sponsored by the Ministry of Communications at KOMET Institut. The course offers certifications from A1 to C1 levels, helping me to improve my proficiency and communication skills in German.
