Smart School Infrastructure Design

This project involved designing a smart school, specifying the locations for data outlets, telecom rooms (MDF and IDF), and the network infrastructure. The design includes strategic placement of security cameras for full coverage, installation of access control systems, and specifying cable trays for a smart, efficient learning environment.

Key Features:

  • Location planning for data outlets
  • Strategic placement of security cameras for full coverage
  • Installation of access control systems
  • Efficient cable tray setup for organized infrastructure
